# push_swap_tester
Tester for 42 school project push_swap


### PREPARE

* Change path to your checker and push_swap programs in top main.py

### USE
```
make
```

### SEE ERRORS
cat errors.txt

### LEAKS
To test leaks you must have valgrind installed

To install valgrind on mac with brew
```
make env_leaks_mac
```
